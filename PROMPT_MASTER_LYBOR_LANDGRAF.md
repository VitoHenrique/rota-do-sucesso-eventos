## 🎨 ESPECIFICAÇÕES DE DESIGN

### Filosofia de Design: **MINIMALISMO SOFISTICADO**

O design segue a abordagem de **minimalismo sofisticado** inspirada em fintechs modernas (Stripe, Wise, Notion). Prioriza clareza, confiança e modernidade através de:

- Espaço em branco generoso (não é vazio, é respiração visual)
- Tipografia refinada e hierárquica
- Paleta de cores restrita e intencional
- Micro-interações suaves e previsíveis
- Foco total no conteúdo e ação do usuário

### Princípios de Design

| Princípio | Descrição |
|-----------|-----------|
| **Espaço como Elemento Ativo** | Muito espaço em branco guia a atenção e cria respiração visual |
| **Tipografia como Hierarquia** | Diferentes pesos e tamanhos criam estrutura sem cores complexas |
| **Cores Intencionais** | Paleta restrita com cada cor tendo propósito específico |
| **Micro-interações Refinadas** | Transições suaves, hover effects sutis, animações que reforçam ações |
| **Profissionalismo Através da Simplicidade** | Menos é mais; cada elemento tem razão de existir |

### Movimento de Design
**Minimalismo Contemporâneo + Design de Fintech**

Inspiração em empresas como Stripe, Wise e Notion que usam espaço, tipografia e cor de forma intencional e elegante para transmitir confiança e modernidade.

## 🔤 TIPOGRAFIA

### Fontes

| Uso | Fonte | Weights | Fallback |
|-----|-------|---------|----------|
| **Headlines (H1-H6)** | Geist | 700, 800 | Inter, sans-serif |
| **Body Text** | Inter | 400, 500, 600 | sans-serif |
| **Código (se necessário)** | Fira Code | 400 | monospace |

**Importação (Google Fonts):**
```html
<link href="https://fonts.googleapis.com/css2?family=Inter:wght@400;500;600;700;800&family=Geist:wght@700;800&display=swap" rel="stylesheet">
```

### Hierarquia Tipográfica

| Elemento | Tamanho | Weight | Line Height | Uso |
|----------|---------|--------|-------------|-----|
| **H1 (Hero)** | 48-60px | 700-800 | 1.2 | Headlines principais, hero |
| **H2 (Section)** | 36-48px | 700 | 1.3 | Títulos de seção |
| **H3 (Subsection)** | 24-32px | 600-700 | 1.4 | Subtítulos, card titles |
| **Body Large** | 18px | 400-500 | 1.6 | Descrições, leads |
| **Body** | 16px | 400-500 | 1.6 | Texto padrão |
| **Body Small** | 14px | 400 | 1.5 | Texto secundário, captions |
| **Label** | 12-14px | 500-600 | 1.5 | Labels, badges |

### Regras Tipográficas

- **Headlines:** Sempre usar Geist, weight 700+, tracking tight
- **Body:** Sempre usar Inter, weight 400-500, line-height 1.6
- **Contraste:** Texto deve ter contraste mínimo de 4.5:1 com background
- **Legibilidade:** Máximo 75 caracteres por linha para body text
- **Espaçamento:** Usar múltiplos de 4px para margins/padding

## ✨ ANIMAÇÕES E INTERAÇÕES

### Velocidades de Transição

| Tipo | Duração | Easing |
|------|---------|--------|
| **Hover Rápido** | 150-200ms | cubic-bezier(0.23, 1, 0.32, 1) |
| **Transição Padrão** | 200-300ms | cubic-bezier(0.23, 1, 0.32, 1) |
| **Entrance** | 300-500ms | cubic-bezier(0.23, 1, 0.32, 1) |
| **Modal/Drawer** | 300-500ms | cubic-bezier(0.23, 1, 0.32, 1) |

### Micro-interações

#### Botões
- **Hover:** 
  - Mudança de cor (para hover color)
  - Scale: 1.02-1.05
  - Duração: 200ms
  
- **Active/Click:**
  - Scale: 0.97
  - Feedback visual imediato
  - Duração: 100ms

#### Links
- **Hover:**
  - Underline aparece
  - Cor muda para primária
  - Duração: 150ms

#### Cards
- **Hover:**
  - Border color muda para primária/30
  - Shadow aumenta
  - Scale: 1.05
  - Duração: 300ms

#### Inputs
- **Focus:**
  - Border color muda para primária
  - Background muda levemente
  - Box-shadow com cor primária
  - Duração: 150ms

### Animações de Entrada

- **Fade-in:** Opacity 0 → 1, duração 300ms
- **Slide-up:** Transform translateY(20px) → 0, duração 300ms
- **Stagger:** 30-80ms entre itens em listas

### Respeitar Preferências do Usuário

```css
@media (prefers-reduced-motion: reduce) {
  * {
    animation: none !important;
    transition: none !important;
  }
}
```

## 📝 NOTAS IMPORTANTES

### Para Aprovação Meta

1. **Consistência é Crítica:** Qualquer variação nas informações da empresa (nome, endereço, telefone) pode resultar em rejeição. Verifique múltiplas vezes.

2. **HTTPS Obrigatório:** Sem HTTPS, a Meta não aprovará. Certifique-se de que o certificado SSL está ativo e válido.

3. **Informações Visíveis:** As informações da empresa devem ser facilmente encontradas. Não coloque em rodapé pequeno ou em página oculta.

4. **Funcionalidade:** O site deve estar totalmente funcional. Erros 404, links quebrados ou páginas em construção resultarão em rejeição.
